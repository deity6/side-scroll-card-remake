# MainMenu 春日树影版素材目录

本目录用于存放主菜单正式运行素材。`main_menu.gd` 会按以下稳定命名自动加载图片；缺图时使用场景内占位显示。

## 背景层

- `main_menu_bg_far.png`：远景森林入口背景，不需要透明。
- `main_menu_tree_body.png`：主树、树干和树影主体。
- `main_menu_fg_leaves.png`：前景枝叶遮挡，建议透明 PNG。

## 父菜单艺术字

每个父菜单艺术字按语言和状态拆分：

- `menu_start_zh_dark.png` / `menu_start_zh_glow.png`
- `menu_start_en_dark.png` / `menu_start_en_glow.png`
- `menu_continue_zh_dark.png` / `menu_continue_zh_glow.png`
- `menu_continue_en_dark.png` / `menu_continue_en_glow.png`
- `menu_settings_zh_dark.png` / `menu_settings_zh_glow.png`
- `menu_settings_en_dark.png` / `menu_settings_en_glow.png`
- `menu_quit_zh_dark.png` / `menu_quit_zh_glow.png`
- `menu_quit_en_dark.png` / `menu_quit_en_glow.png`

`dark` 是融入阴影的常态，`glow` 是悬停、按下或触摸按下时的亮态。

## 彩蛋小游戏素材

小游戏当前已有代码占位。正式素材生成后建议继续使用稳定英文命名，例如：

- `mini_card_projectile.png`：小型光卡。
- `mini_card_trail.png`：发光拖尾。
- `mini_leaf_01.png`、`mini_leaf_02.png`、`mini_leaf_03.png`：小落叶帧。
- `mini_fruit_01.png`、`mini_fruit_02.png`、`mini_fruit_03.png`：小果子帧。
- `mini_eyes_01.png`、`mini_eyes_02.png`、`mini_eyes_03.png`：暗处眼睛帧。

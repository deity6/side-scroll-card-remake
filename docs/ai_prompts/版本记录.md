# 版本记录

## v0.3.3

完成日期：2026-06-26

### 主要内容

- 新增 **SoundManager** 全局音效管理器（Autoload 单例），支持 SFX 播放（随机 pitch）、BGM 淡入淡出预留、播放器池管理。
- 下载并接入 **28 个免费音效素材**（来源 mixkit.co），覆盖 UI/冒险/战斗/奖励四类场景共 31 处调用点。
- 音效映射配置文件 Data/sfx_mapping.json，新增音效只需添加映射和素材文件，无需改代码。
- 主菜单重构：背景从绿色占位恢复黑色，按钮和游戏名居中对齐。
- 彩蛋小游戏代码从 menu_controller.gd 分离到独立脚本 easter_egg_minigame.gd，通过 setup() 注入节点引用。
- 修复 attle_scene.gd 中 _on_card_played 未使用参数警告。
- 清理 58 个临时文件和调试脚本，更新 .gitignore。

### 关键资源路径

- ssets/audio/sfx/ui/ — UI 音效（6 个）
- ssets/audio/sfx/adventure/ — 冒险音效（8 个）
- ssets/audio/sfx/battle/ — 战斗音效（11 个）
- ssets/audio/sfx/reward/ — 奖励音效（3 个）
- Data/sfx_mapping.json — 音效键到文件路径的映射

### 关键场景与脚本

- scripts/sound_manager.gd — SoundManager Autoload 单例
- scripts/easter_egg_minigame.gd — 彩蛋小游戏控制器
- scripts/menu_controller.gd — 主菜单控制器（含音效接入）
- scripts/adventure_scene.gd — 冒险场景（含音效接入）
- scripts/battle_scene.gd — 战斗场景（含音效接入）
- scenes/main_menu.tscn — 主菜单场景（黑色背景 + 居中布局）
- Data/sfx_mapping.json — 音效映射配置

### 已知后续工作

- BGM 素材待后续添加（SoundManager 已预留 BGM 接口，play_bgm() / stop_bgm() 可用）。
- 部分音效可能需要根据实际游戏体验微调替换。

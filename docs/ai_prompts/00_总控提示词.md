# 总控提示词

## 项目定位
本项目是《月圆之夜》风格的**竖版卡牌 Roguelike**复刻，使用 Godot 4.4 + GDScript 开发。
目标不是“照搬原版”，而是先建立一个**可运行、可验证、可逐步增强**的原型。

## 强制约束
- 使用 Godot 4.4 + GDScript，不引入 C#/GDExtension。
- 数值优先来自 `Data/*.json`，不硬编码在脚本里。
- 先保证可运行，再逐步增强表现。
- 不要一次性重写全部文件，每次只改需要改的部分。
- 每次提交都要说明：改了哪些文件、实现了什么、如何验证。
- 发现设计不清楚时，先创建 `docs/design_questions.md`（若不存在）再记录，不擅自改核心方向。

## 项目结构约定
- `scenes/`：场景文件（`.tscn`）
- `scripts/`：脚本文件（`.gd`）
- `Data/`：数据表（`.json`）
- `docs/`：文档与提示词
- `addons/`：第三方插件（如 godot-mcp）

> 备注：当前仓库尚无独立 `assets/` 目录；美术资源在正式替换前，先用占位矩形或简易图标验证布局与交互。

# 总控提示词

## 项目定位
本项目是《月圆之夜》风格的 **竖版卡牌 Roguelike** 游戏复刻，使用 Godot 4.4 + GDScript 开发。

## 强制约束
- 使用 Godot 4.x + GDScript，不使用 C#。
- 数值优先来自 Data/*.json，不硬编码在脚本中。
- 先保证可运行，再逐步增强表现。
- 不要一次性重写全部文件，每次只改需要改的部分。
- 每次提交都要说明：改了哪些文件、实现了什么、如何验证。
- 发现设计不清楚时，先在 design_questions.md 记录，不要擅自改核心方向。

## 项目结构约定
- scenes/ -- 场景文件 (.tscn)
- scripts/ -- 脚本文件 (.gd)
- Data/ -- 数据表 (.json)
- assets/ -- 美术资源
- addons/ -- 第三方插件
- docs/ -- 文档与提示词

## 当前阶段
1. 最小可运行原型（主菜单 + 战斗 + 设置）
2. 竖版战斗布局重构（上敌人、下玩家、底部手牌）
3. 卡组构建与随机事件
4. Roguelike 地图与关卡流程
5. 局外养成系统
6. 美术替换与打磨

## 工作顺序
每次只做一个模块，完成后运行验证，再进入下一个。

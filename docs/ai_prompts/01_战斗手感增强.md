# 战斗手感增强

## 目标
把战斗从「能跑」提升到「打得爽」，让玩家能清晰感知每一次出牌的反馈。

## 输入层
- 拖拽出牌：卡牌从手牌区拖到敌人/自己身上释放
- 点击快速施放：点击手牌自动打到默认目标
- 取消手势：拖拽中松手取消，卡牌飞回手牌区
- 目标选择：多敌人时需先点选目标

## 时间层
- 出牌时序：预动作 -> 飞行/展开 -> 结算 -> 后摇
- 全局时间缩放：出牌瞬间轻微慢动作（hit-stop）
- 敌人行动前有 0.5 秒预告动画

## 反馈层
- 伤害数字弹出：红色数字向上飘出并消失
- 护甲数字弹出：蓝色数字
- 治疗数字弹出：绿色数字
- 手牌数量、牌库/弃牌堆数量实时显示
- 能量消耗动画（能量珠减少）

## 信息层
- 敌人意图图标：攻击（剑）、防御（盾）、技能（星）
- 玩家当前回合可造成的伤害预估
- 下回合抽牌数量提示

## 竖版布局
- 上方：敌人区域（敌人立绘 + 血条 + 意图）
- 中间：玩家区域（玩家立绘 + 血条 + 护甲 + 能量）
- 下方：手牌扇形展开区
- 出牌动画从下往上飞向敌人

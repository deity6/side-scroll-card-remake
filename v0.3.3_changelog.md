# v0.3.3 更新日志

## 主要更新

### 音效系统
- 新增 **SoundManager** 全局音效管理器（Autoload 单例）
  - 支持 SFX 播放（带随机 pitch ±5%）
  - 支持 BGM 淡入淡出（预留接口，素材待后续添加）
  - 每个音效最多同时播放 4 个实例
  - 音效映射配置：Data/sfx_mapping.json
- 下载并接入 **28 个免费音效素材**（来源 mixkit.co）
  - UI 类（6个）：按钮点击、开始游戏、弹窗提示、悬停、返回
  - 冒险类（8个）：卡片选择/消失/吸入、休息恢复、铁匠打铁、商店、宝箱、下一章
  - 战斗类（11个）：抽牌、悬停、出牌、伤害、受击、护甲、治疗、回合、胜负、经验
  - 奖励类（3个）：升级、金币、获得卡牌

### 主菜单重构
- 背景色从绿色占位恢复为黑色
- 菜单按钮和游戏名居中对齐
- 彩蛋小游戏代码分离到 easter_egg_minigame.gd

### 代码优化
- 修复 battle_scene.gd 未使用参数警告
- 移除未实现的 BGM 调用，消除运行时警告
- 清理临时文件和调试脚本

## 文件变更
- 新增：scripts/sound_manager.gd, scripts/easter_egg_minigame.gd, Data/sfx_mapping.json
- 新增：assets/audio/ 目录（28个 MP3 音效素材）
- 修改：project.godot, menu_controller.gd, adventure_scene.gd, battle_scene.gd
- 修改：scenes/main_menu.tscn（黑色背景 + 居中布局）
